# Open Waters

Open source marine software and data — tools to understand and navigate the sea, built through open collaboration.

Hi, I'm Brandon. I started Open Water Software as a home for projects related to marine software and data, hoping it grows into a community where mariners, developers, and ocean enthusiasts come together to build better tools for the sea.

The ocean is a commons. We all depend on it, and we've all benefited from centuries of mariners, hydrographers, and scientists sharing what they learned. Open source works the same way: freely sharing what we build makes the tools better for everyone. Everything here is open, and contributions are welcome.

Learn more at [openwaters.io](https://openwaters.io), or [get in touch](mailto:hello@openwaters.io).

## 🌙 Tides

- [neaps](https://github.com/openwatersio/neaps) — a tide prediction engine
- [tide-database](https://github.com/openwatersio/tide-database) — a public database of tide harmonics
- [signalk-tides](https://github.com/openwatersio/signalk-tides) — a SignalK plugin serving tidal predictions for your vessel's position

## 🗺️ Bathymetry

- [seascape](https://github.com/openwatersio/seascape) — a map of the seafloor built from global open bathymetry
- [crowd-depth](https://github.com/openwatersio/crowd-depth) — collect and share depth soundings from your vessel with crowd-sourced bathymetry programs

## 🧭 Charts & Navigation

- [OpenWaters](https://github.com/openwatersio/OpenWaters) — an experiment in modern marine charts (alpha — not for navigation)
- [enc-tiles](https://github.com/openwatersio/enc-tiles) — tools to produce vector tiles from Electronic Navigational Charts
